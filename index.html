ZPPSU Portal

Student Directory System

A full-featured university student management portal built for Zamboanga Peninsula Polytechnic State University. Features include student record CRUD operations, real-time dashboard statistics, movie browser integration via TMDB API, and a responsive design with a warm maroon and cream color palette.

Key Features
  • Add, edit, and delete student records
  • Dashboard with live statistics (total students, courses, year levels)
  • Movie Browser powered by TMDB API
  • Responsive design with maroon & cream theme
